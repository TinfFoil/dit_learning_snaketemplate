# Snakes on text, a DIT project template example

A small project showcasing a reasonable structure for a python project

## Project setup 

One can follow the [official instructions to setup a virtual environment](https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/)

In summary, the first time a virtual environment has to be installed. 

### Installation

It is advisable that each project has its own virtual environment with its
dependencies (as a result, one can have multiple indepnedent projects/environments)

**GNU Linux / macOS:**

`$ python3 -m venv .venv`

**Windows**

`py -m venv .venv`


### Activation of the virtual environment

To activate the right python (and corresponding dependencies):

**GNU Linux / macOS:**

`source .venv/bin/activate`

**Windows**

`.venv\Scripts\activate`

 ### Deactivation of the virtual environment

 `deactivate`
